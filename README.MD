# empty-project

Empty project.

## Building and running on localhost

First install dependencies:

```
npm install
```

To create a production build:

```
npm run build:product
```

To create a development build:

```
npm run build:dev
```

## Running development

```
npm run start:dev
```

## Testing

To run unit tests:

```
npm run test
```
